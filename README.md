# BinarySearch-Tree

[7, 5, 1, 8, 3, 6, 0, 9, 4, 2] --> Will be used Binary Search-Tree

Let our root be: 6;

                  6       First root: 6
                /   \
               3     8    Second root: 3 & 8
              / \   / \
             2   4 7   9  Third root: 2 & 4 || 7 & 9
            /     \ 
           1       5
